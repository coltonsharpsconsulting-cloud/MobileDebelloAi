## 🧠 Knowledge Bank Philosophy

The goal here is simple: **every insight, every match result, every student breakthrough gets captured and connected.** Nothing lives in your head alone.

```
Every note connects → graph gets richer → patterns emerge → better coaching
```

**The three linking rules:**
1. Every note tags itself (`#vex`, `#student`, `#competition`, etc.)
2. Every note links back to this MOC or a parent note
3. Every note uses `[[double brackets]]` to reference related notes
