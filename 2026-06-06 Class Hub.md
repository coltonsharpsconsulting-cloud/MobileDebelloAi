Built the **Class Hub** (`Scripts/class-hub.js`) — the live operating surface for a single course, slotted above the units on each course's hub note. Shows **this week's lesson** as an editorial card (click → open the week) and the **whole arc** as a clickable build-out strip (each week colored by how built-out it is, the current week outlined). Borrows the shared school-week from any dated curriculum week, so courses without their own dates still know which week is "now."
